# Calculadora de IMC

## Sobre o projeto:

Projetinho simples com a função de calcular seu IMC. Para fazer este cálculo é necessário usar a simples fórmula: peso / altura * altura. 

## Tecnologias usadas:
- Javascript
- HTML
- CSS
- Bootstrap
